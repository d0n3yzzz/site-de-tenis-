 👟 SneakerStore

SneakerStore é um projeto frontend de uma loja virtual de tênis desenvolvido com HTML, CSS e JavaScript puro.

O projeto simula uma experiência moderna de ecommerce sneaker, com foco em design minimalista, responsividade e funcionalidades dinâmicas utilizando `localStorage`.

---
 tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript Vanilla
- LocalStorage

---

 Funcionalidades

 Página Principal
- Hero section moderna
- Grid de produtos responsiva
- Busca de tênis por nome
- Filtro por marcas
- Cards com hover animado
- Contador de itens no carrinho

---

 Página de Produto
- Página dinâmica baseada no ID do produto
- Galeria de imagens
- Seleção de tamanho
- Controle de quantidade
- Adicionar ao carrinho
- Feedback visual ao adicionar produto

---
Carrinho
- Carrinho dinâmico com localStorage
- Atualização automática de subtotal
- Cálculo de frete
- Cálculo do total
- Alteração de quantidade
- Remoção de produtos
- Persistência dos dados mesmo após fechar o navegador
